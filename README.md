# 📊 Data Cleaning & Visualization Project
## 📌 Overview
This project focuses on cleaning raw datasets and generating meaningful visual insights using Python.
## 🚀 Features
- Handle missing values
- Remove duplicates
- Detect and remove outliers
- Data visualization using Matplotlib & Seaborn
## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
